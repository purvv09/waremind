🛡️ Mission: The Secure Data Flow

Think of a hacker 😈 trying to enter a high-security Bank. Most people think they take the shortest route. But they don't.

They look for the "Path of Least Resistance"—the route where the security level (Weights) is the lowest across all nodes.

This project uses Dijkstra's Algorithm to find that hidden weak path so we can fix it before it's too late!
